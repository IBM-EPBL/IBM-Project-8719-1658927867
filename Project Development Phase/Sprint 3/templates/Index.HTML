Handwritten Digit Recognition Web Application



The website is designed to predict the handwritten digit.

The handwritten digit recognition is the ability of computers to recognize human handwritten digits.It is a hard task for the machine because handwritten digits are not perfect and can be made with many different flavors.The handwritten digit recognition is the solution to this problem which uses the image of a digit and recognizes the digit present in the image.


Hence, our handwritten digit recognition web-app is a real-time applications. MNIST data set is widely used for this recognition process and it has 70000 handwritten digits. We use Artificial neural networks particularly cnn to train these images and build a deep learning model. Web application is created where the user can upload an image of a handwritten digit. This image is analyzed by the model and the detected result is returned on to UI
